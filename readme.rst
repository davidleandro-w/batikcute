###################
BATIKCUTE
###################

Sebuah website toko online batik yang dikembangkan menggunakan Framework Codeigniter 3 dan dengan tampilan dari Bootstrap SB Admin 2. Terdapat fitur keranjang, pesan, melihat pesanan, berdasarkan status pesanan, login, registrasi, mengelola database oleh admin, melihat laporan keuangan, mengelola stok, dll.
